Quick Start Guide

1. Include Easy Toast

<link rel="stylesheet" href="/path/to/easy-toast.min.css">
<script src="/path/to/easy-toast.js"></script>

2. Create a Toast

showToast({
    type: 'success',
    title: 'Great job!',
    message: 'Your toast is ready.',
    duration: 3000
});
